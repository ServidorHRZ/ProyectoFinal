Paso 4: Instalación de Visual Studio Code en Fedora

Pasos para instalar Visual Studio Code:

  1. Importar las llaves de Microsoft:
    sudo rpm --import https://packages.microsoft.com/keys/microsoft.asc
  2. Importar el repositorio de VSCode:
    echo -e "[code]\nname=Visual Studio Code\nbaseurl=https://packages.microsoft.com/yumrepos/vscode\nenabled=1\ngpgcheck=1\ngpgkey=https://packages.microsoft.com/keys/microsoft.asc" | sudo tee /etc/yum.repos.d/vscode.repo > /dev/null
  3. Verificar actualizaciones:
    dnf check-update
  4. Instalar Visual Studio Code:
    sudo dnf install code

Pasos para instalar Eclipse:

  1. Primero, asegurarse de tener Java instalado:
    sudo dnf install java-latest-openjdk-devel
  2. Descargar Eclipse desde el sitio oficial:
    wget https://www.eclipse.org/downloads/download.php?file=/technology/epp/downloads/release/2024-03/R/eclipse-java-2024-03-R-linux-gtk-x86_64.tar.gz
  3. Extraer el archivo descargado:
    sudo tar xzf eclipse-java-*-linux-gtk-x86_64.tar.gz -C /opt
  4. Crear un acceso directo para Eclipse:
    echo -e "[Desktop Entry]\nVersion=1.0\nType=Application\nName=Eclipse\nComment=Eclipse IDE\nExec=/opt/eclipse/eclipse\nIcon=/opt/eclipse/icon.xpm\nCategories=Development;IDE;\nTerminal=false" | sudo tee /usr/share/applications/eclipse.desktop

Paso 2: Configuración inicial del sistema durante la instalación

Pasos clave durante la instalación:

  1. Seleccionar idioma y ubicación
  2. Configurar el teclado
  3. Configurar la red:
    Hostname: debian-server Dominio: local
  4. Configurar usuarios y contraseñas
  5. Particionar el disco:
    / (root): 15GB /home: 10GB swap: 2GB

Paso 3: Configuración post-instalación y actualización del sistema

Comandos esenciales post-instalación:

  1. Actualizar lista de paquetes:
    sudo apt update
  2. Actualizar el sistema:
    sudo apt upgrade -y
  3. Instalar herramientas esenciales:
    sudo apt install -y vim net-tools htop
