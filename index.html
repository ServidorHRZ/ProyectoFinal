  3. Verificar actualizaciones:
    dnf check-update
  4. Instalar Visual Studio Code:

Pasos para instalar Eclipse:

  1. Descargar e instalar Java OpenJDK:
    sudo dnf install java-latest-openjdk.x86_64
  2. Instalar OpenJDK para programadores o developments:
    sudo dnf install java-latest-openjdk-devel.x86_64
  3. Descomprimir los .tar.gz:
    tar xzf eclipse-*.tar.gz -C /opt
  4. Movemos la carpeta eclipse para la carpeta opt:
    sudo mv eclipse /opt/
  5. Modificar permisos:
    sudo chmod 755 /opt/eclipse
  6. Escribimos estas líneas:
    [Desktop Entry] Name=Eclipse Type=Application Exec=/opt/eclipse/eclipse Terminal=false Icon=/opt/eclipse/icon.xpm Comment=Integrated Development Environment NoDisplay=false Categories=Development;IDE; Name[en]=Eclipse
  7. Creamos un enlace simbólico para el IDE:
    sudo ln -s /opt/eclipse/eclipse /usr/local/bin/eclipse

4. Instalación y Configuración de Docker

4.1 Instalación de Docker Desktop

Paso 1: Verificar Requisitos

Activar WSL 2:

  1. Abrir PowerShell como administrador y ejecutar:
    wsl --install

    Esto instala la última versión de WSL y configura WSL 2 como predeterminado.

Configurar una distribución de Linux:

  1. Ver distribuciones disponibles:
    wsl --list --online
  2. Instalar una distribución específica:
    wsl --install -d <nombre-de-la-distribución>

Paso 2: Descargar Docker Desktop

  1. Visitar la página oficial de Docker
  2. Descargar la versión compatible con tu sistema operativo

Paso 3: Instalar Docker Desktop

  1. Ejecutar el instalador de Docker Desktop
  2. Aceptar los términos y condiciones
  3. Seleccionar "Enable WSL 2 Features" durante la instalación
  4. Verificar que tu distribución de Linux esté marcada para integración con Docker

Paso 4: Verificar Instalación

  1. Abrir Command Prompt (CMD) o una terminal en tu distribución de Linux en WSL
  2. Ejecutar el comando:
    docker --version

    Esto mostrará la versión de Docker instalada.

4.2 Creación de imagen Docker

Paso 1: Crear y configurar el Dockerfile

Comandos utilizados:

  1. Crear directorio del proyecto:
    mkdir proyecto-debian-docker
  2. Navegar al directorio:
    cd proyecto-debian-docker
  3. Crear archivo Dockerfile:

6. Instalación y Configuración de Kubernetes

6.1 Instalación en Docker Desktop

Paso 1: Habilitar Kubernetes en Docker Desktop

  1. Abrir Docker Desktop
  2. Ir a Configuración (Settings)
  3. Seleccionar "Kubernetes" en el menú lateral
  4. Marcar la casilla "Enable Kubernetes"
  5. Hacer clic en "Apply & Restart"

Paso 2: Verificar la instalación

Comandos para verificar la instalación:

  1. Verificar la versión del cliente:
    kubectl version --client
  2. Verificar el estado del clúster:
    kubectl cluster-info
  3. Listar los nodos:
    kubectl get nodes
